# Pytorchnanigans

Experimenting with Pytorch, magic, and max.

# Setup

1. Install [magic](https://docs.modular.com/magic/)

2. Install dependencies
```
magic install
```

# Run

To run the notebook either:

1. Enter the shell

```
magic shell
```

2. Run the notebook

```
jupyter notebook notebooks/scratchpad.ipynb
```

or 

1. Run via the task
```
magic run scratchpad
```
